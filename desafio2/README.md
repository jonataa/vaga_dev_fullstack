# Desarmando a Bomba

## Requisitos
python >= 3.5

## Como usar
```shell
$ python antibombas.py VERMELHO IAJNLITLUNAYDHFAA
BOOOMMM!!
$ python antibombas.py AZUL EFDLUMHBNDRRTM
BOOOMMM!!
$ python antibombas.py VERDE ZTRAHDSICFQH
BOOOMMM!!
$ python antibombas.py AMARELO QPSKXDLPWFLAAKHY
Bomb defused :)
```
